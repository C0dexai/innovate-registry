Anatomy of a Workflow: CI/CD Automation

1

Build Stage

Agent: ANDIE

Clones the repository from the specified branch and builds a new container image.

2

Test Stage

Agent: DAN

Executes automated unit and integration tests against the newly built container.

3

Deploy Stage

Agent: STAN

Deploys the validated container image to the staging environment for review.

4

Notify Stage

Agent: GUAC

Sends a completion notification to the Operator, confirming successful deployment.
